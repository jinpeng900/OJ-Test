# OJ 在线判题系统 v0.2

## 版本概述

**版本号**: v0.2  
**发布日期**: 2025年  
**技术栈**: C++17 / MySQL / CMake 3.10+ / OpenSSL  

v0.2 版本在 v0.1 基础上实现了用户模块的完整数据库交互、题目浏览功能优化、CLI 界面清屏规范以及代码结构优化。

| 功能模块           | 具体功能            | 说明                              |
| ------------------ | ------------------- | --------------------------------- |

## 核心类接口详解

### 1. User 类（已完整实现）

**头文件**: `include/user.h`  
**源文件**: `src/user.cpp`

#### 构造函数
```cpp
User(DatabaseManager* db);
```

#### 方法

| 方法                  | 签名                                                                           | 功能         | 数据库交互         |
| --------------------- | ------------------------------------------------------------------------------ | ------------ | ------------------ |
| `login`               | `bool login(const string& account, const string& password)`                    | 用户登录     | ✅ 验证账号密码     |
| `register_user`       | `bool register_user(const string& account, const string& password)`            | 用户注册     | ✅ 插入新用户       |
| `change_my_password`  | `bool change_my_password(const string& old_pwd, const string& new_pwd)`        | 修改密码     | ✅ 验证旧密码并更新 |
| `list_problems`       | `void list_problems()`                                                         | 查看题目列表 | ✅ 查询 problems 表 |
| `view_problem`        | `void view_problem(int id)`                                                    | 查看题目详情 | ✅ 查询指定题目     |
| `submit_code`         | `void submit_code(int problem_id, const string& code, const string& language)` | 提交代码     | ⏳ 待实现           |
| `view_my_submissions` | `void view_my_submissions()`                                                   | 查看我的提交 | ⏳ 待实现           |
| `is_logged_in`        | `bool is_logged_in() const`                                                    | 获取登录状态 | -                  |
| `get_current_account` | `string get_current_account() const`                                           | 获取当前账号 | -                  |

#### SHA256 密码哈希

```cpp
static string sha256(const string &password)
```

使用 OpenSSL EVP 接口计算 SHA256 哈希，简化版实现便于初学者理解。

## CLI 清屏规范

### 清屏实现

```cpp
void clear_screen()
{
    cout << "\033[2J\033[3J\033[H";
    cout.flush();
}
```

| 转义序列  | 功能             |
| --------- | ---------------- |
| `\033[2J` | 清除屏幕         |
| `\033[3J` | 清除滚动缓冲区   |
| `\033[H`  | 移动光标到左上角 |

### 清屏场景汇总

| 场景         | 用户模式 | 管理员模式 |
| ------------ | -------- | ---------- |
| 进入模式     | ✅        | ✅          |
| 登录成功     | ✅        | -          |
| 查看题目列表 | ✅        | ✅          |
| 查看题目详情 | ✅        | ✅          |
| 查看我的提交 | ✅        | -          |
| 修改密码     | ✅        | -          |
| 发布新题目   | -        | ✅          |
| AI 助手      | ✅        | -          |
